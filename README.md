# Memory Match Game

A fun interactive memory card game where players flip cards and match pairs using logic and memory. The goal is to complete the board with the fewest moves possible.

---

## Live Demo [Memory Game](https://harpreetkaur2201.github.io/Memory-Match-Game/)

## Features
- Card flip matching gameplay
- Random shuffled deck every game
- Move counter for score tracking
- Next target card preview system
- Win popup with animated icons
- Restart button to replay instantly
- Responsive and clean UI

## Built With
- HTML5
- CSS3
- JavaScript 
- Font Awesome Icons

## How to Play
1. Look at the next target icon
2. Click cards to reveal icons
3. Match the correct icon shown in the preview
4. Match all pairs to win the game
5. Try to finish with the least moves

## Restart Game
You can restart anytime using:
- Restart button in UI
- Browser refresh
